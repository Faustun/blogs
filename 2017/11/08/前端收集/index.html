    • git clone https://github.com/hjzheng/front-end-collect
    • cd front-end-collect
    • bower install
    • 放入你喜欢的web容器,访问index.html即可
    • 你也直接可以访问: http://get-set.cn/front-end-collect/
    • 支持Chrome, Firefox and IE10&11以上浏览器
